\doxysubsection{Descripción detallada}
Class that represents a particle in Particle Swarm Optimization (PSO). Each particle has\+:
\begin{DoxyItemize}
\item A current state (position).
\item A personal best state (p\+Best).
\item A velocity vector. 
\end{DoxyItemize}
