\doxysubsection{Descripción detallada}
Class that implements a classic Tabu Search generator.

It maintains\+:
\begin{DoxyItemize}
\item A current reference solution.
\item A candidate selection rule.
\item A Tabu list of forbidden solutions (via {\ttfamily \doxylink{}{Tabu\+Solutions}}). 
\end{DoxyItemize}
